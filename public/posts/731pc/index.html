資訊科技概論：通往未來的基石
#

大家常說的「電腦課」，在高中正式名稱為 資訊科技概論。
但在新竹高中，這門課從來就不是「上網課」或「自由活動課」。

曾聖超老師（Tseng Sheng-Chau）擁有國立臺灣科技大學博士學位，長期指導竹中軟體研究社，課程內容橫跨 作業系統、網路原理、開源軟體與資訊倫理，是許多竹中人踏入資訊工程領域的第一步。

重要 這不只是一堂課，而是資訊素養與工程思維的起點。

一、電腦課不是「上網衝浪課」
#

每年第一堂課，總會有同學滿懷期待地問：

「老師，我們可以自由上網嗎？」
「以前國中電腦課都讓我們自己玩耶！」

而老師的回答，幾乎成為竹中電腦教室的經典語錄：

「抱歉！老師也認真準備很多有趣的內容，沒辦法讓你們隨性玩。」

這句話的意思很清楚：
電腦不是拿來打發時間的工具，而是需要被理解的系統。

許多所謂的「自由操作」，最後只會變成遊戲或社群網站。曾老師選擇用更高的標準，要求學生真正理解電腦在做什麼。

二、課程為什麼這麼「硬」？(十年前的硬喔喔喔喔喔喔~)
#

和多數只教介面操作的電腦課不同，曾老師的課程從一開始就要求學生用鍵盤思考。

🐧 UNIX / Linux 作業系統
#

學生會實際接觸 UNIX / Linux 環境，而不是只停留在 Windows 桌面：

  • 檔案系統與目錄結構
  • 權限概念與使用者角色
  • FTP 檔案傳輸（搭配 FileZilla）

並實際使用：

  • vi
  • joe

這些在工程界仍被廣泛使用的文字編輯器。

🎨 開源軟體實作訓練
#

課程強調 Open Source 精神，讓學生理解「工具不是唯一解」。

  • GIMP：影像處理、班徽與宣傳圖設計
  • OpenOffice：文件、試算表與簡報製作

重點不在於「會不會按」，而是理解軟體背後的概念與邏輯。

🌐 網路與網頁基礎
#

在 Web 介面泛濫的時代，課程反而回到最原始的基礎：

  • IP 位址與網路拓樸
  • BBS 文字介面
  • HTML 原始碼結構

學生不是用「產生器」，而是直接理解網頁是怎麼被寫出來的。

三、灑掃應對進退：資訊課也教態度
#

資訊科技概論不只教技術，也教態度與責任感。

有句閩南語說得很實在：

「洗面洗耳空，掃地掃壁角」

意思是：事情不只要做，還要做對、做到位。

教室公約與習慣
#

  1. 準時上課：尊重課堂與他人
  2. 關機還原：為下一位使用者著想
  3. 設備回報：發現異常必須紀錄

這些看似瑣碎的規定，其實正是工程精神的延伸。

四、為什麼這堂課會被記得這麼久？
#

很多學長回頭看才發現：

  • 當年學的指令 現在還在用
  • 當年覺得老派的內容 反而最耐用
  • 筆記規定要寫，沒寫去寫閱讀心得!

介面會改、工具會換，但底層邏輯不會。
這也是為什麼曾老師的課，被稱為竹中資訊教育的基石。

十年前的硬喔喔喔喔喔喔~
#

曾老師的教材名不虛傳，擁有最先進十年不變的教學方法 讓學生學會

出去可以用十年前的軟體更別家公司競爭

未來學會使用搜尋引擎來找查資料
學習如何使用電子郵件傳訊息!

這些十年前的東西雖然教法不便，但是總是讓大家喜歡
